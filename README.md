# 🏏 From Bell Curves to Batting Scores  
### Statistical Analysis of Cricket Performance

---

## 📌 Project Overview

This project explores cricket batting performances using statistical concepts such as:

- Normal Distribution
- Skewness
- Standard Normal Variate (Z-Score)
- Empirical Rule
- Distribution Analysis

The goal is to understand whether cricket batting runs follow a predictable statistical pattern or not.

---

## 🎯 Problem Statement

Cricket performances often appear random, but do they follow a statistical structure?

This project investigates:
- How batting runs are distributed
- Whether the data follows a normal distribution
- How consistent players are across innings
- Whether extreme performances create skewness

---

## 📊 Dataset Overview

- Player-wise batting runs dataset
- Limited top players included
- Each row represents an innings

### Features:
- Player_Name
- Runs

---

## 📈 Visualizations Used

- Histogram + KDE Plot
- Mean Line Overlay
- Player-wise Boxplot
- Z-Score Distribution (optional extension)

---

## 🧠 Key Concepts Covered

### 📌 Normal Distribution
Understanding how cricket runs behave in a bell curve pattern.

### 📌 Skewness
Measuring asymmetry in batting performances due to high scoring innings.

### 📌 Z-Score
Standardizing player performances to identify outliers.

### 📌 Empirical Rule
Understanding data spread within 1σ, 2σ, and 3σ ranges.

---

## 📊 Key Insights

- Most cricket runs are concentrated around the mean
- Data shows slight positive skewness due to high scoring innings
- Players show varying levels of consistency
- Extreme performances act as statistical outliers
- Real-world cricket data is close to but not perfectly normal

---

## 📌 Final Conclusion

Cricket batting performance is not purely random — it follows a structured statistical pattern with natural variability, consistency clusters, and occasional extreme performances that shape the distribution.

---

## 🔥 Final Thought

> “Cricket is not just a game of runs — it's a game of patterns hidden inside numbers.”
